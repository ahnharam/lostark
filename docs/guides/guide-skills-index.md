## 개요
- 목적: 프로젝트 내 Codex 스킬을 빠르게 찾기

## 업데이트 규칙
- 스킬 추가/삭제 시 `docs/README.md`와 `docs/dev-quickstart.md`의 문서 목록을 갱신하세요.
