# Lostark Frontend UX Guides

## Workflow
1. Read `frontend/docs/UX_OVERVIEW.md` for the app-level flow and component map.
2. Read `frontend/docs/ARK_GRID_GUIDE.md` when the Ark Grid UI is in scope.
3. Read `frontend/docs/CHARACTER_RANKING_GUIDE.md` when ranking is in scope.
4. Align terminology with existing component and file names.
5. Call out loading/error/empty states and interaction rules.

## Output expectations

- Keep UI state descriptions concise and actionable.
- Link to the most relevant doc(s) for follow-up work.
