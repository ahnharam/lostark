# Lostark Documentation Guidelines

## Workflow
1. Read `docs/documentation-guidelines.md`.
2. Determine the correct category and file prefix.
3. Ensure kebab-case naming and folder placement.
4. Apply the standard header template for new docs when needed.
5. Update required indexes after adding or moving docs.
6. Prefer updating existing docs over creating new ones.

## Output expectations
- Suggest file name(s) and folder path.
- Provide a short checklist of required updates.
- Keep language and path formatting consistent with the guidelines.
