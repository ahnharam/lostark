# WCAG 3.0: Readable Blocks of Text (초안 요약)

> **출처**: https://www.w3.org/TR/wcag-3.0/#readable-blocks-of-text  
> **상태**: Developing (초안)

## 핵심 요구사항 문장
“The default/authored presentation of blocks of text meets the corresponding values for the content’s language (or the language of the ones listed with the most similar orthography).”

## 편집자 노트 (요약)
- Readable blocks of text / Readable text style은 **일반적 사용 관행** 기반
- 보완(Supplemental) 항목은 **가독성 연구** 기반
- 일부 언어는 **추가 연구 필요**
- 표의 지표는 **예시값**으로 표기됨

## 표출값 (English 예시)

| Characteristic | Value (English example) |
|---|---|
| Inline margin | (미기재) |
| Block margin | ≥ 0.5em around paragraphs |
| Line length | 30–100 characters |
| Line height | 1.0 – paragraph separation height |
| Justification | Left aligned or Justified |

## 적용 메모
- 언어별 값이 없으면 **유사한 orthography 언어** 기준을 사용
- 초안 값은 **프로젝트 기준으로 확정/검증 필요**
